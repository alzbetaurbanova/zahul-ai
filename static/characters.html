Import Character Card

Supported formats

.pngSillyTavern / TavernAI character card (metadata embedded in PNG)
.jsonNative format (below) or Pygmalion / TavernAI JSON export
.jsoncJSON with comments — same as .json but supports // annotations

Native JSON template

{
  "name": "CharacterName",
  "data": {
    "persona": "Character description...",
    "instructions": "Respond only in English.",
    "avatar": "https://...",
    "info": "Short description shown on card.",
    "temperature": null,
    "history_limit": null,
    "max_tokens": null
  },
  "triggers": ["trigger1", "trigger2"]
}

Leave temperature, history_limit, max_tokens as null to use global config values. Triggers can be added after import.
